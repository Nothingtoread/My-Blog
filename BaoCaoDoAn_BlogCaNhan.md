## CHƯƠNG 2: CƠ SỞ LÝ THUYẾT

### 2.1. React - Thư viện JavaScript cho giao diện người dùng

#### 2.1.1. Giới thiệu về React

React là thư viện JavaScript mã nguồn mở được phát triển bởi Facebook (Meta) năm 2013, được sử dụng để xây dựng giao diện người dùng, đặc biệt là các Single Page Applications (SPA).

**Đặc điểm nổi bật:**
- **Component-based:** Xây dựng UI từ các components độc lập, có thể tái sử dụng
- **Declarative:** Mô tả UI nên trông như thế nào, React lo việc cập nhật
- **Virtual DOM:** Tối ưu hóa hiệu suất render
- **One-way data flow:** Dữ liệu chỉ đi theo một chiều, dễ debug

#### 2.1.2. JSX (JavaScript XML)

JSX là cú pháp mở rộng của JavaScript, cho phép viết HTML trong JavaScript:

```javascript
const element = <h1>Hello, World!</h1>;

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}
```

#### 2.1.3. Components

Components là khối xây dựng cơ bản của React:

**Function Components:**
```javascript
function Home() {
  return (
    <div className="home">
      <h1>Xin chào!</h1>
    </div>
  );
}
```

**Props (Properties):**
```javascript
function BlogPost({ title, content }) {
  return (
    <article>
      <h2>{title}</h2>
      <p>{content}</p>
    </article>
  );
}
```

#### 2.1.4. React Hooks

Hooks cho phép sử dụng state và lifecycle trong function components:

**useState - Quản lý state:**
```javascript
const [selectedPdf, setSelectedPdf] = useState(null);
```

**useEffect - Side effects:**
```javascript
useEffect(() => {
  document.title = `Blog - ${postTitle}`;
}, [postTitle]);
```

**useCallback - Memoize functions:**
```javascript
const handleClose = useCallback(() => {
  setSelectedPdf(null);
}, []);
```

### 2.2. React Router - Quản lý điều hướng

React Router là thư viện chuẩn cho routing trong React applications.

**Cấu hình routes:**
```javascript
<Router>
  <Routes>
    <Route path="/" element={<Home />} />
    <Route path="/blog" element={<Blog />} />
    <Route path="/blog/:id" element={<BlogPost />} />
    <Route path="/projects" element={<Projects />} />
  </Routes>
</Router>
```

**Navigation:**
```javascript
import { Link } from 'react-router-dom';

<Link to="/blog">Blog</Link>
```

### 2.3. CSS và Responsive Design

#### 2.3.1. CSS Modules

Mỗi component có file CSS riêng để tránh xung đột:
```
Home.js
Home.css
```

#### 2.3.2. Flexbox và Grid

**Flexbox** - Layout một chiều:
```css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

**Grid** - Layout hai chiều:
```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 2rem;
}
```

#### 2.3.3. Media Queries

Responsive design cho các kích thước màn hình:
```css
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
```

### 2.4. Node.js và NPM

#### 2.4.1. Node.js

Node.js là runtime environment cho phép chạy JavaScript trên server, được sử dụng để:
- Chạy development server
- Build production bundle
- Quản lý dependencies

#### 2.4.2. NPM (Node Package Manager)

NPM quản lý các packages/libraries:
```json
{
  "dependencies": {
    "react": "^19.2.3",
    "react-dom": "^19.2.3",
    "react-router-dom": "^7.11.0"
  }
}
```

### 2.5. Create React App

Create React App là công cụ chính thức để khởi tạo React project với cấu hình tối ưu:

- Testing setup với Jest

```javascript

const handleViewPdf = (fileName, imageFileName, certName) => {
  setSelectedPdf({ fileName, imageFileName, certName });
  document.body.style.overflow = 'hidden';
};

// Đóng modal
const handleCloseModal = useCallback(() => {
  setSelectedPdf(null);
  document.body.style.overflow = 'auto';
}, []);

// Download PDF
const handleDownload = (fileName, event) => {
  event.stopPropagation();
  const link = document.createElement('a');
  link.href = `/certificates/${fileName}`;
  link.download = fileName;
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
};
```

4. **Keyboard Support:**
```javascript
useEffect(() => {
  const handleEscKey = (event) => {
    if (event.key === 'Escape' && selectedPdf) {
      handleCloseModal();
    }
  };
  
  if (selectedPdf) {
    document.addEventListener('keydown', handleEscKey);
  }
  
  return () => {
    document.removeEventListener('keydown', handleEscKey);
  };
}, [selectedPdf, handleCloseModal]);
```
